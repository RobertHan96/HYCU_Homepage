HTML의 역사

  • 1980년 - 팀버너스리에 의해 최초 개발
  • 1989년 - 하이퍼텍스트 개념 도입
  • 1990년 - 최초의 웹브라우저 개발
  • 1993년 - SGML, 최초의 HTML 기준 확립
  • 1995년 - HTML 2.0 규격 개발
  • 2000년 - HTML 4.0.1이 국제표준으로 설정 됨
  • 2008년 - 다양한 기능이 포함된 HTML5가 W3C에 의해 고안됨
