HTML

웹피이지 구현을 위해 필요한 기술들을 알아봅니다.

HTML, CSS, JavaSrcpit를 통해 웹페이지를 꾸미고, 서버와 클라이언트간에 상호작용을 할 수 있습니다. 각 언어별 특징은 다음과 같습니다.

  • HTML(Hyper Text Markup Language)
    • web page를 만들기 위해 필요한 일종의 언어로, HTML을 통해 웹페이지의 구조를 작성함
  • CSS (Cascading Style Sheets)
    • HTML 문서의 스타일을 설명하는 역할, CSS3에서는 각종 애니메이션 효과도 가능해짐
  • JavaSrcpit
    • 인터프리터 방식의 웹프로그래밍 언어로, HTML문서의 동작을 제어해서 동적인 페이지를 만들어줌
